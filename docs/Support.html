  • What you expected to happen and what actually happened.

Common Questions

Can I use both Reolink and generic RTSP cameras?

Yes. Security Center supports both. Reolink cameras can use JPG or RTSP, while generic streams use RTSP.

Why is my screen black during quiet hours?

That is expected. Quiet hours intentionally black out the screen and pause polling, streaming, and availability checks until the configured end time.

Can I save my setup and move it to another device?

Yes. The app supports JSON import and export using the same structure it uses for its stored configuration.

Can I make a camera appear in more than one grid?

Yes. Grid assignments are independent, so the same camera can be reused across saved layouts.

Why does a generic RTSP camera fail validation?

The full RTSP URL must be valid and reachable. Check the scheme, host, port, credentials, and stream path.

Troubleshooting

Snapshot view says unavailable

Check the host, username, password, channel, and whether the camera should use HTTP or HTTPS for Reolink JPG snapshots.

Live stream says unavailable

Confirm the RTSP endpoint is reachable on the expected port and that the stream path and credentials are correct.

Sidebar status is not updating

Availability checks pause during quiet hours and return false for disabled cameras. If neither applies, review the camera connection details.

A grid cell is empty

Open the saved grid and assign a camera to that cell. Empty cells intentionally show a Select Camera control instead of a placeholder feed.

Changes were not saved after closing the editor

App settings save on Done, while camera changes save on Save. Closing the sheet without saving keeps the edit local and can discard it.
